Overview

This section describes the smart agriculture rover's goal to automate field monitoring using sensors and motor controls.

Controllers

ESP32 for rover control and ESP8266 for sensor and wireless operations.

Sensors

DHT11, BMP180, Rain Sensor, Soil Moisture Sensor, Ultrasonic and IR sensors for environmental and obstacle detection.

How It Works

The ESP32 and ESP8266 communicate to control the rover and irrigation based on sensor feedback.

Web Access

Access rover and sensor dashboards via ESP IP-based web servers.

Power

Powered by 12V lithium battery, rechargeable via adapter or solar panel.
