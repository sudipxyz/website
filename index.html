Smart Agriculture Rover

Automated Field Monitoring and Irrigation System

 Components

Overview

The Smart Agriculture Rover is designed to enhance modern farming practices by automating key field operations. This rover navigates through the field autonomously, collecting real-time environmental and soil data. Its core purpose is to assist farmers with real-time monitoring of agricultural conditions such as temperature, humidity, rainfall, atmospheric pressure, and soil moisture levels. Based on this data, it can make intelligent decisions, such as notifying farmers or triggering irrigation through a wireless control node.

The system is fully solar-powered and capable of being monitored and controlled remotely via a web interface. Using GPS and motion sensors, it maintains balance and direction. A variety of sensors and embedded microcontrollers allow this rover to function as a mobile weather station and smart irrigation advisor.

Component List

  • ESP32 Microcontroller: Main control unit for rover movement and real-time decision making.
  • ESP8266 Microcontroller (Sensor Node): Collects weather and soil data using various sensors.
  • ESP8266 Microcontroller (Control Node): Activates water pump based on soil moisture data.
  • MPU6050: Gyroscope and accelerometer module for balancing and orientation tracking.
  • GPS Module: Provides real-time location tracking of the rover.
  • Ultrasonic Sensor: Detects obstacles in front of the rover for collision avoidance.
  • IR Sensor: Detects obstacles at the back of the rover.
  • DHT11: Measures temperature and humidity.
  • BMP180: Measures atmospheric pressure.
  • Rain Sensor: Detects presence of rainfall.
  • Soil Moisture Sensor: Monitors soil water content.
  • 12V DC High Torque Motors: Provides drive power to the rover wheels.
  • L298N Motor Driver: Controls motor direction and speed using ESP32.
  • 12V Lithium Battery: Power source for the entire system.
  • Solar Panel: Used to recharge the battery and power the rover in the field.

Sensors

The Smart Agriculture Rover integrates a wide array of sensors to gather critical data:

  • DHT11: Measures temperature and relative humidity to assess air conditions in real time.
  • BMP180: Captures atmospheric pressure, useful for weather prediction and altitude estimation.
  • Rain Sensor: Detects precipitation to prevent irrigation during rainfall.
  • Soil Moisture Sensor: Measures the water content of the soil to decide if irrigation is required.
  • Ultrasonic Sensor: Senses obstacles in front to avoid collisions during navigation.
  • IR Sensor: Identifies obstacles behind the rover for reverse safety and maneuvering.

How It Works

The system architecture includes a mobile rover unit powered by an ESP32 controller. The ESP32 handles the mechanical movement of the rover, processes input from obstacle sensors (ultrasonic and IR), maintains balance using the MPU6050, and tracks position via GPS.

A separate ESP8266-based sensor node gathers climate and soil data and sends this information wirelessly to another ESP8266 control node. Based on the soil moisture data, this node decides whether to activate a water pump. Farmers can monitor and control the rover and sensor units using a mobile-responsive web interface hosted directly by the microcontrollers.

Web Access

The system features a web-based interface hosted on the ESP32 and ESP8266 devices. Users can access the rover's controls or view sensor data by entering the respective device's IP address into a prompt on the main interface.

The ESP32 interface allows remote navigation of the rover, while the ESP8266 displays environmental readings. This decentralized web system enables easy access from any smartphone, tablet, or computer without needing an external server or internet.

Power

The Smart Agriculture Rover is powered by a 12V lithium-ion battery that ensures consistent performance in field conditions. The battery supplies energy to the microcontrollers, motors, and sensors.

A solar panel is integrated into the system to charge the battery during daylight, making the system self-sustaining. Charging can also be done via an AC adapter when solar energy is insufficient. This hybrid approach ensures uninterrupted operation in remote agricultural areas.
